(1) What is the independent variable? What is the dependent variable?

In this project, the independent variables are the words which are congruent with the color or not. The depentdent variables are the response time.

(2) What is an appropriate set of hypotheses for this task? What kind of statistical test do you expect to perform? Justify your choices.

In this project, an appropriate null hypotheses H0 would be there is no difference in population means of the response time under Congruent and Incongruent tests, and the alternative hypothesis HA would be there exists a difference in the population means. In math, it can be expressed as: H0: C = I HA: C != I Here C is the population mean of the response time for Congruent test and I is the population mean of the response time for Incongruent test.

We can use two-tailed t-test to help us examine since the alternative hypothesis is C != I. As we can find from the given data, the variances for congruent and incongruent are unknown, and the sample size is small. The difference of response time between the congrudent and incongrudent would follow a student's t-distribution. Based on these reasons, t-test would be a better choice rather than z-test.

# Perform the statistical test here
# Run t-test
d = np.diff([Congruent, Incongruent], axis = 0)
d_mean = np.mean(d)
d_std = np.sqrt(np.sum((d - d_mean)**2)/(len(Congruent)-1))
d_SE = d_std/np.sqrt(len(Congruent))
T = d_mean/d_SE
print('By formula, t_score=', T)

# By stats package 
t_score, p_value = stats.ttest_rel(Congruent, Incongruent)
print('By scipy package, t_score, p_value =', t_score, p_value)

# critical value for t-test
# For a 95% confidence interval with two-tail, the critical value will be
t_critical = stats.t.ppf(0.975, len(Congruent)-1)
print('t_critical value =', t_critical)
